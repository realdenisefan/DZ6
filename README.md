# DZ6 изменено
ДЗ 6 КисвянцевДС
- Импортируется класс `datetime` из модуля `datetime`.
- Определяется функция `parse_newspaper_date(newspaper, date_str, date_format)`, которая:
  - получает название газеты, строку даты и строку формата;  
  - пытается преобразовать строку с помощью `datetime.strptime`;  
  - обрабатывает возможные ошибки `ValueError` и выводит сообщение об ошибке.
- Создаётся словарь `newspapers`, где для каждой газеты указана:
  - строка с датой;  
  - соответствующая строка формата (`%A, %B %d, %Y`, `%A, %d.%m.%y`, `%A, %d %B %Y`).
- В цикле перебираются все газеты, вызывается функция `parse_newspaper_date`, и для успешно разобранных дат результат печатается в формате `Газета - YYYY-MM-DD HH:MM:SS`.
